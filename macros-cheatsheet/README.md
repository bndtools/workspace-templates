# BND Macros Cheatsheet

This template fragment provides a comprehensive reference for bnd macros through a well-documented `bnd.bnd` file.

## What's Included

This template copies a project `org.bndtools.cheatsheet` to your workspace containing:

- **bnd.bnd** - Comprehensive examples of all major bnd macros with descriptions
- **example.txt** - Sample file for testing file-based macros (cat, base64, digest)
- **README.md** - basic information how to use the template fragment
